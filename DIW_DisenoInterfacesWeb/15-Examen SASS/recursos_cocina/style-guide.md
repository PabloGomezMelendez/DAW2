# Sitio web Escuela de cocina

## Guía de estilos

### Tipografía

- Tipo de letra principal: *Roboto 300 y 400*

- Dimensiones (si se define un font-size de 62.5% en el elemento html):
  - Textos
    - h1: 4.4rem
    - h2: 3.4rem
    - h3: 2.4rem
    - p: 1.6rem
    - resto: 1.6rem
  - Altura de línea: 1.8 para los párrafos y 1.2 para los titulares

### Colores

- Principal: #f46669
- Secundario: #c7c57d
- Gris: #f5f5f5
- Negro: #000000
- Blanco: #ffffff
